## Layout

| Folder | Course / Topic |
| --- | --- |
| `cpp/` | C++ labs, quizzes, midterm, homeworks, and HackerRank exercises |
| `cv/` | Computer Vision problem sets (ps1, ps2, ps3) |
| `db/` | Database / blockchain project (`chain` Hardhat workspace, `backend` Node service) |
| `dld/` | Digital Logic Design notes, reference PDFs, and project files |
| `java/` | Java labs, homeworks, midterm/final exercises |
| `os/` | Operating Systems homework |
| `physics/` | Physics study materials, in-class slides, old exams |
| `py/` | Introductory Python labs, quizzes, homeworks, final |
| `src/` | Shared assets |
| `web/` | Web project (HTML / CSS / JS) |

## Notes

All identifying information (names, handles, IDE paths, build artifacts, environment files) has been removed. Java packages are normalized to `com.example`. The repository is published for reference only.
